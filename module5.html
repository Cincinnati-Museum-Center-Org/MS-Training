OneDrive Basics
Save to OneDrive ‚Äî Not Your Desktop
‚ñº
Files saved to your desktop only exist on that computer. OneDrive saves files to the cloud so you can access them from anywhere, share them instantly, and never lose them if your computer crashes.

Key Takeaways

  • OneDrive = your personal cloud storage in Microsoft 365
  • SharePoint = your team or department's shared cloud storage
  • Files in OneDrive/SharePoint auto-save every few seconds
  • Use the OneDrive folder on your PC ‚Äî it syncs automatically

Sharing Links & Permissions
Share a Link ‚Äî Not an Attachment
‚ñº
When you email an attachment, everyone gets a separate copy ‚Äî and edits diverge immediately. A sharing link points to the one cloud version, so everyone always sees the latest file.

Key Takeaways

  • Right-click a file in OneDrive ‚Üí Share ‚Üí Copy Link
  • Can View ‚Äî recipient can read but not change anything
  • Can Edit ‚Äî recipient can modify the file directly
  • "Anyone with the link" ‚Äî risky if forwarded; prefer "Specific people"
  • Set an expiry date for sensitive links

Co-authoring
Collaborate in Real Time with Co-authoring
‚ñº
When a file is saved in OneDrive or SharePoint, multiple people can edit it at the same time. You can see each other's cursors in real time ‚Äî no more "who has the file open?" emails.

Key Takeaways

  • Co-authoring works in Word, Excel, and PowerPoint online and desktop
  • Colored cursors show who is editing where in real time
  • Use @mention in a comment to notify a specific colleague
  • If someone is offline, changes sync when they reconnect

File Naming & Etiquette
Naming Conventions + When to Comment vs Edit
‚ñº
Good file names make things findable. Clear norms about when to comment vs edit directly prevent collaboration friction and protect document integrity.

Key Takeaways

  • Good name: 2025-Q3-Budget-Draft.xlsx ‚Äî date first, descriptive, no spaces
  • Bad name: Final_REAL_v3_USE THIS ONE.xlsx
  • Comment = question or suggestion that others should respond to
  • Edit directly = only when you have explicit permission to make changes
  • Version history in OneDrive means you can always roll back
