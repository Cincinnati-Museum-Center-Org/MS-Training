Styles & Layout
Use Styles Instead of Manual Formatting
‚ñº
Heading 1, Heading 2, and Normal styles keep your document visually consistent and enable automatic Table of Contents. Manual formatting (bold, font size) looks consistent now but breaks down when you make changes.

Key Takeaways

  • Apply Heading 1 for main sections, Heading 2 for sub-sections
  • Modify a Style once and it updates everywhere ‚Äî huge time saver
  • Consistent styles also improve accessibility for screen readers

Layout & Structure
Page Layout, Lists, Tables & Page Numbers
‚ñº
Good document structure makes reading effortless. Master page margins, line spacing, bulleted and numbered lists, simple tables for data, headers/footers, and automatic page numbers.

Key Takeaways

  • Layout ‚Üí Margins: 1" standard for most professional documents
  • Line spacing 1.15 or 1.5 is easier to read than single-spaced
  • Tables work better than tab-spaced columns for structured info
  • Insert ‚Üí Page Number ‚Üí choose position to add automatic page numbers

Track Changes & Comments
Collaborate Without Confusion
‚ñº
Track Changes records every edit so reviewers can see what changed. Comments let you ask questions or flag things without editing the document itself. Together they make document reviews clear and professional.

Key Takeaways

  • Review ‚Üí Track Changes: turns on red markup for every edit
  • Accept or Reject changes one at a time or all at once
  • Comments: select text ‚Üí Review ‚Üí New Comment (or Ctrl+Alt+M)
  • Comments = suggestions/questions; Track Changes = actual edits

Version Control
Saving to OneDrive/SharePoint to Avoid Version Conflicts
‚ñº
Emailing Word documents back and forth creates "Final_v3_REAL_final2.docx" chaos. Saving to OneDrive or SharePoint means one version, real-time co-authoring, and automatic version history.

Key Takeaways

  • File ‚Üí Save As ‚Üí OneDrive (or SharePoint) ‚Äî one click to save in the cloud
  • Multiple people can edit the same document at the same time
  • File ‚Üí Info ‚Üí Version History to see and restore older versions
  • Share a link, not an attachment ‚Äî always the latest version
