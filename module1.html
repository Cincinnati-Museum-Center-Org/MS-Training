Writing Emails
Subject Lines, CC vs BCC, Attachments & @mentions
‚ñº
Learn the habits that make emails clearer and more actionable: strong subject lines, knowing when to CC vs BCC, attaching files correctly, and using @mentions to call out individuals.

Key Takeaways

  • Subject line = action + topic (e.g. "Review needed: Q3 report")
  • CC = informational; BCC = hidden recipient (use sparingly)
  • Use Insert ‚Üí Attach File for OneDrive links, not desktop drag-and-drop
  • @mention someone in the body to alert them even if they're on CC
